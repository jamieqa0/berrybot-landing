- **서비스명:** 베리봇 (BerryBot)
- **형태:** 원페이지 웹사이트 (랜딩 페이지)
- **주제:** 딸기를 직접 세척·다듬어주는 **가정용 스마트 로봇** 구독 서비스

---

## 베리봇 — 로봇 정체성

베리봇은 **딸기 손질에 특화된 가정용 스마트 로봇**이다. 구독자의 가정에 베리봇을 배송·설치하여, 집에서 직접 딸기를 넣으면 세척·다듬기·꼭지 제거까지 자동으로 처리해준다.

| 항목 | 내용 |
|------|------|
| 외형 | 딸기 모양 몸통, 초록 잎 왕관, 노란 씨앗 무늬, 팔·다리·발 장착 |
| 작업 방식 | 딸기 투입 → 자동 세척 → 품질 검수 → 꼭지 제거 → 서빙 트레이 출력 |
| 표정 변화 | 컷마다 감정 표현 (신남 / 하트눈 / 집중 / 뿌듯) |
| 서비스 포지션 | "집에서 로봇이 직접 손질"이라는 가정용 편의 가전으로서의 차별화 강조 |

### 실제 로봇 외형 참고 (제품 설계 기반)

베리봇은 주방 조리대에 올려두는 **컴팩트한 스마트 가전** 형태.

- **본체:** 무광 화이트 또는 파스텔톤 유선형 바디
- **투명 돔:** 상단은 강화유리/투명 폴리카보네이트 — 작동 과정을 실시간으로 관찰 가능
- **크기:** 10인용 전기밥솥 수준

**핵심 메커니즘:**

| 부위 | 명칭 | 역할 |
|------|------|------|
| 상단 입구 | 호퍼(Hopper) | 씻지 않은 딸기를 한꺼번에 투입 |
| 중앙부 | 클린 존(Clean Zone) | 미세 버블 + 초음파로 잔류 농약 제거 (3단 세척) |
| 작업대 | 트리밍 데크(Trimming Deck) | AI 카메라로 꼭지 위치 감지 후 소프트 그리퍼로 정밀 제거 |
| 하단 출구 | 서빙 트레이(Serving Tray) | 손질 완료 딸기가 담겨 나오는 접시 거치대 |

구독 = 베리봇 로봇 이용권. 상위 플랜일수록 처리 가능한 베리류 종류가 확대된다.

| 플랜 | 가격 | 지원 과일 | 로봇 기능 |
|------|------|-----------|-----------|
| **라이트** 🍓 | 월 9,900원 | 딸기 | 기본 세척 + 꼭지 제거 |
| **홈** 🫐 | 월 19,900원 | 딸기 + 블루베리 | 세척·다듬기·꼭지 제거 전 과정 + AI 손질 품질 보증 |
| **패밀리** 🍒 | 월 34,900원 | 딸기 + 블루베리 + 라즈베리 + 체리 | 전 과정 + 우선 처리 예약 + 과일 쉐이크 모드 |

- 모든 플랜: 첫 달 50% 할인 (체험 혜택)
- 언제든지 플랜 변경·해지 가능

## 4컷 만화 콘티

> 캐릭터: 베리봇 — 딸기 모양 몸통, 동그란 눈, 팔·다리 달린 귀여운 로봇

### 1컷 — "넣어볼까요!"
- 베리봇이 딸기 바구니를 두 팔로 들고 있는 장면
- 말풍선: "냠냠냠... 신선한 베리 넣어줘요!" / 효과음: 반짝반짝 ✨
- 배경: 주방, 딸기 무늬 앞치마

### 2컷 — "깨끗이 씻어줄게!"
- 수도꼭지 아래서 딸기를 씻는 장면, 눈이 하트 모양
- 말풍선: "먼지야 안녕~" / 효과음: 쏴아아~ 💧

### 3컷 — "예쁘게 다듬어볼게!"
- 돋보기를 들고 딸기를 꼼꼼히 살펴보는 장면
- 말풍선: "흠... 완벽해!" / 효과음: 반짝! ⭐

### 4컷 — "짜잔~ 드세요!"
- 꼭지 제거된 딸기를 그릇에 담아 두 손으로 내미는 장면, 뿌듯한 표정
- 말풍선: "꼭지까지 완벽하게! 베리봇 임무 완료~" / 효과음: 짜잔~! 🎉

## 디자인 요구사항

- **스타일:** 4컷 만화/웹툰 느낌
- **테마:** 딸기 컨셉 (빨강/초록/흰색 컬러, 씨앗 패턴, 딸기 아이콘)
- **캐릭터:** 베리봇이 주인공 — "로봇이 직접 손질"이라는 점을 카피·UI 전반에 명시
- **일러스트:** 직접 제작 (`assets/illustrations/panel1~4.svg`)
- **표현 금지:** 배송·수량·신선도 보증 등 과일 배송 서비스로 읽힐 수 있는 표현
